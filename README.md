# AI-Powered Audio & Video Translation System (In Progress)

## Overview

This project is an AI-powered translation system that converts audio and video content into translated text across multiple languages.

The application extracts audio from media files, processes transcripts, translates content using Google's Gemini API, and generates multilingual outputs.

## Technologies

- Python
- Gemini API
- FFmpeg
- Git
- FastAPI (future enhancement)

## Features

- Audio extraction from videos
- Automated transcription workflow
- AI-powered translation
- Multiple language support
- Prompt-engineered translation pipeline

## Workflow

Video File
→ FFmpeg
→ Transcript Generation
→ Gemini Translation
→ Multilingual Output

- Speech-to-text integration using Whisper
